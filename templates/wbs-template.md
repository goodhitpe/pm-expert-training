## WBS 검증 체크리스트

- [ ] **100% 규칙**: 상위 항목이 하위 항목의 100%를 포함하는가?
- [ ] **상호 배타성**: 하위 항목 간 중복이 없는가?
- [ ] **적절한 수준**: 작업이 관리 가능한 크기로 분해되었는가?
- [ ] **산출물 지향**: 각 작업이 명확한 산출물을 가지는가?
- [ ] **명확성**: 작업명이 이해하기 쉽고 명확한가?
- [ ] **완전성**: 모든 프로젝트 작업이 포함되었는가?
