## 💼 비즈니스 임팩트

### ROI 분석

**투자 비용**: ₩55,020,000 (개선 프로젝트)

**예상 수익 증가**:
1. **수강료 인상** 가능
   - 현재: ₩3,000,000/인
   - 개선 후: ₩3,500,000/인 (+17%)
   - 차이: ₩500,000/인

2. **등록자 증가**
   - 현재: 20명/기수 × 4기수/년 = 80명
   - 개선 후: 25명/기수 × 4기수/년 = 100명 (+25%)
   - 증가: 20명/년

3. **재등록/추천 증가**
   - 만족도 향상 → 입소문 → 신규 등록 5명/년

**연간 수익 증가**:
```
(100명 × ₩500,000) + (20명 × ₩3,500,000) + (5명 × ₩3,500,000)
= ₩50,000,000 + ₩70,000,000 + ₩17,500,000
= ₩137,500,000/년
```

**ROI**: (₩137.5M - ₩55M) / ₩55M = **150%**

**Payback Period**: 5개월
