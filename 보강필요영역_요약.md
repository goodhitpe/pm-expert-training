## 🔴 최우선 보강 영역 (2주 안에 완료)

### 1. 사전 학습 자료 12개 추가 ⭐⭐⭐⭐⭐

**문제점**:
- 현재 Week 13-16만 사전 학습 자료 있음 (4개)
- Week 1-12는 사전 학습 자료 없음 (12개 부족)
- 일관성 부족으로 학습자 혼란

**영향**:
- 어려운 개념을 사전 준비 없이 수업 → 이해도 저하
- 학습 곡선 급격 → 중도 포기 위험
- 특히 Week 6 (회계), Week 9 (애자일) 사전 이해 필수

**해결책**:
각 주차별 `prerequisite.md` 파일 추가 (5-8페이지, 20-30분 학습)

**우선순위별 작성 순서**:
1. Week 6 (원가관리) - 회계 기초 용어 ⭐ 가장 시급
2. Week 9 (애자일) - 애자일 마인드셋
3. Week 1 (PM 개요) - PM 기초 개념
4. Week 2 (PM 기초) - PMBOK 용어
5. Week 10 (PM 도구) - 도구 개요
6. 나머지 Week 3-5, 7-8, 11-12

**소요 시간**: 60시간 (주차당 5시간)

**템플릿 구조**:
```markdown
# Week X 사전 학습: [주제명]

> 학습 시간: 20-30분
> 목적: Week X 수업을 위한 필수 기초 개념

## 📚 이 주차를 위한 준비
- 주차 개요 및 필요성

## 핵심 개념 5-7개
- 각 개념: 정의, 예시, PM 관점 설명

## ✅ 자가 진단 체크리스트
- [ ] 개념 1 이해
- [ ] 개념 2 이해
...

## 📖 추천 학습 자료
- 영상 링크
- 읽을거리
```

**예시**: Week 6 사전 학습
```
1. 회계 기초 용어 (원가, 예산, 수익)
2. EVM이란? (획득가치관리 개념)
3. CV, SV, CPI, SPI 용어 미리보기
4. 프로젝트 예산 vs 회사 예산
5. 비용 추정의 중요성
```

**예상 효과**:
- 수업 이해도 +20%
- 학습 만족도 +15%
- 점수 향상: +2.5점

### 2. GitHub 실습 가이드 추가 ⭐⭐⭐⭐⭐

**문제점**:
- 개발자와 협업 필수 도구인데 가이드 없음
- Week 10 (PM 도구)에서 언급만 하고 실습 없음
- Week 13 (소프트웨어 공학)에서 Git 설명하지만 환경 설정 없음

**영향**:
- 학습자가 Git/GitHub 못 쓰면 실습 불가
- 개발자와 협업 능력 저하
- 모의 프로젝트 진행 어려움

**해결책**:
`guides/github-setup-guide.md` 작성 (15-20페이지)

**포함 내용**:
1. GitHub 계정 생성 (무료)
2. Git 설치 (Windows/Mac)
3. Repository 생성
4. 기본 명령어 (clone, add, commit, push)
5. Branch 생성 및 관리
6. Pull Request (PR) 만들기
7. Code Review 기초
8. GitHub Issues 활용
9. **스크린샷 20-30개 포함** (단계별)

**소요 시간**: 8시간

**예상 효과**:
- 실습 참여율 +30%
- 개발자 협업 능력 향상
- 점수 향상: +1.0점

### 3. SQL 실습 환경 가이드 ⭐⭐⭐⭐

**문제점**:

**영향**:

- SQL 쿼리 작성 능력 부족

   - SQLite Online (추천)
   - DB Fiddle
   - SQL Try Editor
2. MySQL Workbench 설치 (선택, 로컬 환경)
3. 샘플 데이터베이스 로드
4. 기본 쿼리 10개 실습 (SELECT, WHERE, JOIN)
5. Week 14 과제 준비

**소요 시간**: 8시간

**예상 효과**:
- SQL 실습 완료율 +40%
- 데이터베이스 이해도 향상
- 점수 향상: +0.8점

**예상 효과**:
- 클라우드 실습 참여율 +35%
- AWS 이해도 대폭 향상
- 점수 향상: +1.0점

- 클라우드 마이그레이션 체크리스트

**Week 16 추가 내용** (10KB):
- HTTP/HTTPS 상세 (메서드, 상태코드)

- 네트워크 보안 강화 (SSL/TLS, VPN)
- CDN, Load Balancer
- Postman으로 API 호출 실습

**소요 시간**: 20시간 (각 10시간)

- 종합 점수: +1.2점
