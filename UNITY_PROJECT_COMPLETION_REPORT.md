## 🏗️ 기술 아키텍처

### Unity 기술 스택
- **Unity 버전**: Unity 2022 LTS
- **아키텍처**: MVC 패턴
- **데이터**: JSON + Addressables
- **UI**: Unity UI + TextMesh Pro
- **마크다운**: Markdig 라이브러리
- **애니메이션**: DOTween
- **비동기**: UniTask

### 시스템 레이어
```
┌─────────────────────────────────┐
│   Presentation Layer (UI)       │
├─────────────────────────────────┤
│   Business Logic Layer          │
├─────────────────────────────────┤
│   Data Access Layer             │
├─────────────────────────────────┤
│   Data Layer (JSON)             │
└─────────────────────────────────┘
```

### 핵심 시스템
1. **GameManager**: 전체 게임 관리
2. **DataManager**: 데이터 로딩/저장
3. **SceneController**: 씬 전환
4. **EventBus**: 이벤트 통신
5. **ProgressTracker**: 진행률 추적
6. **DecisionEngine**: 의사결정 처리
