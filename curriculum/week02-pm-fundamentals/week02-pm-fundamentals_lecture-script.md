### [00:15-02:00] 섹션 1: 5대 프로세스 그룹 (105분)

#### [00:15-00:30] 1.1 PMBOK 소개 및 프로세스 그룹 개요 (15분)

**강의 내용**:
```
📖 PMBOK이란?
"PMBOK은 Project Management Body of Knowledge의 약자로,
PMI(Project Management Institute)가 제정한 프로젝트 관리 표준입니다."

💡 왜 PMBOK을 배워야 하나?
1. 전 세계 표준 언어
   - "뉴욕의 PM과 서울의 PM이 같은 용어로 소통"
2. 체계적인 프레임워크
   - "감으로 하는 것이 아닌 방법론"
3. PMP 자격증 기반
   - "PM 커리어에 필수"

🔄 5대 프로세스 그룹 전체 그림
화이트보드에 큰 흐름도 그리기:

    시작 → [착수] → [계획] → [실행] → 종료
                         ↓      ↑
                    [감시/통제]

"착수부터 종료까지 순차적으로 진행하지만,
실행과 감시/통제는 동시에 반복적으로 일어납니다."

📊 각 프로세스 그룹의 시간 배분
- 착수: 5%
- 계획: 25%
- 실행: 50%
- 감시/통제: 15%
- 종료: 5%

"실행이 절반을 차지하지만, 계획이 없으면 실행도 없습니다!"
```

**교수법**:
- 화이트보드에 큰 그림 그리며 설명
- 각 프로세스 그룹의 비중 시각화
- 실제 프로젝트 진행 흐름과 연결

- 예산: 5억 원

**강의 내용**:

📚 10대 지식영역이란?

"프로세스 그룹이 '언제'를 다룬다면,
지식영역은 '무엇을'을 다룹니다."

📊 프로세스 그룹 vs 지식영역
         착수  계획  실행  감시통제  종료
통합       ●     ●     ●      ●      ●
범위            ●            ●
일정            ●            ●
원가            ●            ●
품질            ●     ●      ●
자원            ●     ●      ●
의사소통         ●     ●      ●
리스크          ●            ●
조달            ●     ●      ●
이해관계자  ●     ●     ●      ●

"49개 프로세스 = 5개 그룹 × 10개 영역 매트릭스"

1️⃣ 통합 관리 (Integration Management)

🎯 역할: "모든 영역을 조율하는 오케스트라 지휘자"

주요 프로세스:

📈 EVM (Earned Value Management)
"프로젝트 성과를 측정하는 기법"

PV (Planned Value): 계획 가치
EV (Earned Value): 획득 가치
AC (Actual Cost): 실제 비용

성과 지표:
- SV (Schedule Variance) = EV - PV
- CV (Cost Variance) = EV - AC
- SPI (Schedule Performance Index) = EV / PV
- CPI (Cost Performance Index) = EV / AC

예시:
PV = 5천만원 (50% 완료 계획)
EV = 4천만원 (40% 실제 완료)
AC = 4.5천만원 (실제 지출)

SV = 4천 - 5천 = -1천만원 (지연)
CV = 4천 - 4.5천 = -500만원 (초과)
SPI = 4천/5천 = 0.8 (20% 지연)
CPI = 4천/4.5천 = 0.89 (11% 초과)

💡 해석
SPI < 1: 일정 지연
CPI < 1: 예산 초과
→ 조치 필요!
