### 2. 원가 유형

#### 2.1 직접 원가 vs 간접 원가
**직접 원가 (Direct Costs):**
- 특정 프로젝트에만 귀속
- 예: 프로젝트 팀원 급여, 전용 장비

**간접 원가 (Indirect Costs):**
- 여러 프로젝트에 공통 배분
- 예: 사무실 임대료, 관리 부서 비용

#### 2.2 고정 원가 vs 변동 원가
**고정 원가 (Fixed Costs):**
- 작업량과 무관하게 일정
- 예: 계약 컨설턴트 비용

**변동 원가 (Variable Costs):**
- 작업량에 비례하여 변동
- 예: 시간당 인건비, 소모품

#### 2.3 매몰 원가 (Sunk Costs)
- 이미 지출되어 회수 불가능한 비용
- 의사결정 시 고려하지 않음

### 3. 원가 산정 기법

#### 3.1 유사 산정 (Analogous Estimating)
- 과거 유사 프로젝트 데이터 활용
- 빠르고 저렴하지만 정확도 낮음
- 초기 단계에 적합

#### 3.2 모수 산정 (Parametric Estimating)
- 통계적 모델 사용
- 예: 개발 라인당 비용 × 총 라인 수
- 정확한 데이터 필요

#### 3.3 상향식 산정 (Bottom-Up Estimating)
- 최하위 작업부터 산정 후 합산
- 가장 정확하지만 시간 소요
- 상세 계획 단계에 적합

#### 3.4 3점 산정
- 낙관적(O), 가능성 높음(M), 비관적(P)
- 기대값 = (O + 4M + P) / 6

#### 3.5 예비비 (Reserve)
**우발 예비비 (Contingency Reserve):**
- 알려진 리스크 대응
- PM 권한으로 사용

**관리 예비비 (Management Reserve):**
- 미지의 리스크 대응
- 경영진 승인 필요

### 4. 예산 수립

#### 4.1 원가 집계
- 활동별 원가를 작업 패키지로 합산

#### 4.2 원가 기준선 (Cost Baseline)
```
원가
 ↑
 │        /-------------- 관리 예비비

```

- 누적 원가를 시간에 따라 표시

### 5. EVM (Earned Value Management)

#### 5.1 EVM 핵심 개념
- **PV (Planned Value)**: 계획된 가치
- **EV (Earned Value)**: 획득 가치
- **AC (Actual Cost)**: 실제 원가

#### 5.2 EVM 계산

**분산 (Variance):**
- **CV (Cost Variance)** = EV - AC
  - 양수: 예산 대비 절감
  - 음수: 예산 초과
  
- **SV (Schedule Variance)** = EV - PV
  - 양수: 일정 앞섬
  - 음수: 일정 지연

**지수 (Index):**
- **CPI (Cost Performance Index)** = EV / AC
  - > 1: 효율적 (1원으로 1원 이상의 가치)
  - < 1: 비효율적
  
- **SPI (Schedule Performance Index)** = EV / PV
  - > 1: 일정 앞섬
  - < 1: 일정 지연

**예측 (Forecast):**
- **EAC (Estimate at Completion)**: 완료 시 예상 총 원가
  - EAC = BAC / CPI (전형적인 편차가 계속될 경우)
  - EAC = AC + (BAC - EV) (일회성 편차일 경우)
  
- **ETC (Estimate to Complete)**: 완료까지 필요한 원가
  - ETC = EAC - AC
  
- **VAC (Variance at Completion)**: 완료 시 예상 분산
  - VAC = BAC - EAC

- **TCPI (To-Complete Performance Index)**: 목표 달성 필요 성과
  - TCPI = (BAC - EV) / (BAC - AC)

#### 5.3 EVM 예제

**프로젝트 정보:**
- BAC (예산): 100,000원
- 현재 시점: 6개월차
- PV: 50,000원
- EV: 45,000원
- AC: 52,000원

**계산:**
- CV = 45,000 - 52,000 = -7,000원 (예산 초과)
- SV = 45,000 - 50,000 = -5,000원 (일정 지연)
- CPI = 45,000 / 52,000 = 0.87 (비효율적)
- SPI = 45,000 / 50,000 = 0.90 (일정 지연)
- EAC = 100,000 / 0.87 = 114,943원
- VAC = 100,000 - 114,943 = -14,943원

**해석:**
- 예산 7,000원 초과, 일정 지연 상태
- 완료 시 약 15,000원 초과 예상
- 시정 조치 필요

## 📝 핵심 용어
- **EVM (Earned Value Management)**: 성과 가치 관리
- **PV (Planned Value)**: 계획 가치
- **EV (Earned Value)**: 획득 가치
- **AC (Actual Cost)**: 실제 원가
- **CPI (Cost Performance Index)**: 원가 성과 지수

- **EAC (Estimate at Completion)**: 완료 시 예상 원가
- **BAC (Budget at Completion)**: 완료 시 예산
