### 2. 원가 유형

#### 2.1 직접 원가 vs 간접 원가
**직접 원가 (Direct Costs):**
- 특정 프로젝트에만 귀속
- 예: 프로젝트 팀원 급여, 전용 장비

**간접 원가 (Indirect Costs):**
- 여러 프로젝트에 공통 배분
- 예: 사무실 임대료, 관리 부서 비용

#### 2.2 고정 원가 vs 변동 원가
**고정 원가 (Fixed Costs):**
- 작업량과 무관하게 일정
- 예: 계약 컨설턴트 비용

**변동 원가 (Variable Costs):**
- 작업량에 비례하여 변동
- 예: 시간당 인건비, 소모품

#### 2.3 매몰 원가 (Sunk Costs)
- 이미 지출되어 회수 불가능한 비용
- 의사결정 시 고려하지 않음

### 3. 원가 산정 기법

#### 3.1 유사 산정 (Analogous Estimating)
- 과거 유사 프로젝트 데이터 활용
- 빠르고 저렴하지만 정확도 낮음
- 초기 단계에 적합

#### 3.2 모수 산정 (Parametric Estimating)
- 통계적 모델 사용
- 예: 개발 라인당 비용 × 총 라인 수
- 정확한 데이터 필요

#### 3.3 상향식 산정 (Bottom-Up Estimating)
- 최하위 작업부터 산정 후 합산
- 가장 정확하지만 시간 소요
- 상세 계획 단계에 적합

#### 3.4 3점 산정
- 낙관적(O), 가능성 높음(M), 비관적(P)
- 기대값 = (O + 4M + P) / 6

#### 3.5 예비비 (Reserve)
**우발 예비비 (Contingency Reserve):**
- 알려진 리스크 대응
- PM 권한으로 사용

**관리 예비비 (Management Reserve):**
- 미지의 리스크 대응
- 경영진 승인 필요

### 4. 예산 수립

#### 4.1 원가 집계
- 활동별 원가를 작업 패키지로 합산

#### 4.2 원가 기준선 (Cost Baseline)

### 5. EVM (Earned Value Management)

#### 5.1 EVM 핵심 개념
- **PV (Planned Value)**: 계획된 가치
- **EV (Earned Value)**: 획득 가치
- **AC (Actual Cost)**: 실제 원가

#### 5.2 EVM 계산

**분산 (Variance):**
- **CV (Cost Variance)** = EV - AC
  - 양수: 예산 대비 절감
  - 음수: 예산 초과
  
- **SV (Schedule Variance)** = EV - PV
  - 양수: 일정 앞섬
  - 음수: 일정 지연

**지수 (Index):**
- **CPI (Cost Performance Index)** = EV / AC
  - > 1: 효율적 (1원으로 1원 이상의 가치)
  - < 1: 비효율적
  
- **SPI (Schedule Performance Index)** = EV / PV
  - > 1: 일정 앞섬
  - < 1: 일정 지연

**예측 (Forecast):**
- **EAC (Estimate at Completion)**: 완료 시 예상 총 원가
  - EAC = BAC / CPI (전형적인 편차가 계속될 경우)
  - EAC = AC + (BAC - EV) (일회성 편차일 경우)
  
- **ETC (Estimate to Complete)**: 완료까지 필요한 원가
  - ETC = EAC - AC
  
- **VAC (Variance at Completion)**: 완료 시 예상 분산
  - VAC = BAC - EAC

- **TCPI (To-Complete Performance Index)**: 목표 달성 필요 성과
  - TCPI = (BAC - EV) / (BAC - AC)

#### 5.3 EVM 예제

**프로젝트 정보:**
- BAC (예산): 100,000원
- 현재 시점: 6개월차
- PV: 50,000원
- EV: 45,000원
- AC: 52,000원

**계산:**
- CV = 45,000 - 52,000 = -7,000원 (예산 초과)
- SV = 45,000 - 50,000 = -5,000원 (일정 지연)
- CPI = 45,000 / 52,000 = 0.87 (비효율적)
- SPI = 45,000 / 50,000 = 0.90 (일정 지연)
- EAC = 100,000 / 0.87 = 114,943원
- VAC = 100,000 - 114,943 = -14,943원

**해석:**
- 예산 7,000원 초과, 일정 지연 상태
- 완료 시 약 15,000원 초과 예상
- 시정 조치 필요

#### 3. EVM을 활용한 클라우드 비용 통제

**프로젝트: 6개월 개발 + 12개월 운영**

**기준선 (BAC - Budget at Completion):**
```
개발 비용 (6개월):
- 인건비: $300,000
- 개발 환경 (Dev/Staging): $3,000/월 × 6 = $18,000
소계: $318,000

운영 비용 (12개월):
- 프로덕션 인프라: $795/월 × 12 = $9,540
- 지원 인력 (0.5 FTE): $60,000
소계: $69,540

총 예산 (BAC): $387,540
```

**Month 3 (50% 진행 시점) 측정:**

```
PV (Planned Value):
- 개발 3개월 완료: $318,000 × 50% = $159,000
- 인프라 3개월: $3,000 × 3 = $9,000
PV = $168,000

EV (Earned Value):
- 실제 완료된 작업의 예산 가치
- 개발 진척률: 45% (일정 약간 지연)
- EV = $318,000 × 45% + $9,000 = $152,100

AC (Actual Cost):
- 실제 지출된 비용
- 인건비: $155,000 (계획대로)
- 인프라: $11,000 (계획 $9,000 초과)
AC = $166,000

=== 성과 분석 ===

CPI (Cost Performance Index):
CPI = EV / AC = $152,100 / $166,000 = 0.92
→ 예산 대비 92% 효율 (8% 초과 지출 중)

SPI (Schedule Performance Index):
SPI = EV / PV = $152,100 / $168,000 = 0.91
→ 일정 대비 91% 진행 (9% 지연 중)

CV (Cost Variance):
CV = EV - AC = $152,100 - $166,000 = -$13,900
→ $13,900 예산 초과

SV (Schedule Variance):
SV = EV - PV = $152,100 - $168,000 = -$15,900
→ $15,900 상당의 작업 지연

EAC (Estimate at Completion):
EAC = BAC / CPI = $387,540 / 0.92 = $421,239
→ 예상 최종 비용: $421,239 (초과 $33,699)

ETC (Estimate to Complete):
ETC = EAC - AC = $421,239 - $166,000 = $255,239
→ 남은 작업 예상 비용

TCPI (To-Complete Performance Index):
TCPI = (BAC - EV) / (BAC - AC)
     = ($387,540 - $152,100) / ($387,540 - $166,000)
     = $235,440 / $221,540 = 1.06
→ 예산 내 완료하려면 앞으로 106% 효율 필요 (어려움)
```

**조치 사항:**
```
1. 인프라 비용 초과 원인 분석
   → 발견: Dev 환경에서 불필요한 대형 인스턴스 사용
   → 조치: t3.large → t3.medium 다운사이징 ($500/월 절감)

2. 개발 일정 지연 대응
   → 크래싱: 개발자 1명 추가 (2개월)
   → 비용: $20,000
   → 효과: SPI 0.91 → 1.0 회복

3. 클라우드 비용 모니터링 강화
   → AWS Cost Explorer 일일 리뷰
   → 예산 알람 설정 (주간 $750 초과 시)
   → Trusted Advisor 권장사항 적용
```

## 📝 핵심 용어
- **EVM (Earned Value Management)**: 성과 가치 관리
- **PV (Planned Value)**: 계획 가치
- **EV (Earned Value)**: 획득 가치
- **AC (Actual Cost)**: 실제 원가
- **CPI (Cost Performance Index)**: 원가 성과 지수
- **SPI (Schedule Performance Index)**: 일정 성과 지수
- **EAC (Estimate at Completion)**: 완료 시 예상 원가
- **BAC (Budget at Completion)**: 완료 시 예산
